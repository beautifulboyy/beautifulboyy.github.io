Date和Calendar的使用

发表于2021-07-25|更新于2021-08-02|内置类的使用
|字数总计:2k|阅读时长:8分钟|阅读量:

Date

  1. api:java.util.Date[C] - 使用这个类之前,必须要导包import
  2. 这是java中提供的关于日期的处理,内置的对象类型.它本身是一个过时的类
  3. 里面提供了很多已经过时的方法.
  4. 目标:掌握这个类中提供的方法

构造方法

构造方法 - 创建这个类的对象的方式

在使用api时首先观察该类提供的方法是静态的还是动态的,发现Date类中提供的方法几乎都是非静态的,所以必须创建该类的对象从而调用方法.

构造方法通过new关键字调用

一个小练习:计算三天之后的此时此刻

常用方法

由api知方法几乎都是非静态方法,需要**日期对象.方法名()**进行调用

  1. long getTime();返回当前日期的毫秒数的表现形式

    获取当前系统时间的毫秒数.距离1970年1月1日

  2. get/set方法 - 过时的

    • void setYear(int year); //year-1900
    • void setMonth(int month); //month的范围[0,11]
    • void setDate(int date); //设置今天是几号
    • void setHours(int hours); //设置小时
    • void setMinutes(int minutes); //设置分钟
    • void setSeconds(int seconds); //设置秒钟

  3. setTime()方法,通过设置毫秒数重新设置Date

Calendar

java.util.Calendar是一个抽象类,它不能被new

常用方法

  1. static Calendar getInstance();

  2. 设置日历字段

  3. 单独针对单个字段一一进行赋值

  4. 获取日历字段

总结

  1. 毫米数 -> Date

    • Date提供的构造方法

    • Date提供的setTime(long 毫秒数)方法

  2. Date -> 毫米数

    Date提供的getTime()方法

  3. Calendar -> Date

    Calendar提供的getTime()方法

  4. Date -> Calendar

    Calendar提供的setTime(Date d)方法
